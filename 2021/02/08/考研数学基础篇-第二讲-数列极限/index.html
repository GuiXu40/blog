考研数学基础篇-第二讲-数列极限

发表于 2021-02-08 分类于 考研 ， 数学

引言

极限： 是一个无限趋近的过程

定义

设 $ { \left\{ {x\mathop{{}}\nolimits_{{n}}} \right\} }$ 为一数列，若存在常数a，对于任意的 ${ \varepsilon > 0}$ (不论它多么小)，总存在正整数N，使得当n>N时， ${{ \left| {x\mathop{{}}\nolimits_{{n}}-a} \right| } < \varepsilon }$ 恒成立，则称数a是数列$ { \left\{ {x\mathop{{}}\nolimits_{{n}}} \right\} }$ 的极限，或者称数列$ { \left\{ {x\mathop{{}}\nolimits_{{n}}} \right\} }$ 收敛于a，记作

$${\mathop{{lim}}\limits_{{n \xrightarrow {} \infty }}\mathop{{x}}\nolimits_{{n}}\text{ }=\text{ }a\text{ }= > \text{ } \forall \varepsilon > 0, \exists N \in \mathop{{N}}\nolimits_{{+}},\text{当}n > N\text{时}\text{，}\text{恒}\text{有}{ \left| {\mathop{{x}}\nolimits_{{n}}-a} \right| } < \varepsilon }$$

注： 这就是${ \varepsilon -N}$ 语言

注二：数列收敛于其子列收敛的关系

  • 子列的定义：从数列中选取无穷多项，并按照原来的先后顺序组成新的数列，则称新的数列为原数列的子列

定理： 若数列 $ { \left\{ {x\mathop{{}}\nolimits_{{n}}} \right\} }$ 收敛，则其任何子列 $ { \left\{ {x\mathop{{}}\nolimits_{{n1}}} \right\} }$ 也收敛，且二则极限相等。

使用： 可以证明数列发散。对于一个数列 $ { \left\{ {x\mathop{{}}\nolimits_{{n}}} \right\} }$ ，如果能找到一个发散的子列，则原数列一定发散；如果能找到两个收敛的子列 $ { \left\{ {x\mathop{{}}\nolimits_{{n2}}} \right\} }$ 和 $ { \left\{ {x\mathop{{}}\nolimits_{{n2}}} \right\} }$ ，但是他们收敛到不同极限，则原数列也一定发散

收敛数列的性质

定理一：唯一性

给出数列 $ { \left\{ {x\mathop{{}}\nolimits_{{n}}} \right\} }$ ，若${\mathop{{{\mathop{{lim}}\limits_{{n- > \infty }}\mathop{{x}}\nolimits_{{n}}=a}}}\nolimits_{{}}}$ (存在)， 则a是唯一的。

使用： 判断某个点是否有极限，如果不唯一，说明极限是不存在的

定理二：有界性

若数列 $ { \left\{ {x\mathop{{}}\nolimits_{{n}}} \right\} }$ 极限存在，则说明 $ { \left\{ {x\mathop{{}}\nolimits_{{n}}} \right\} }$ 有界

补充： 什么是有界：

函数的有界性是数学术语。 设函数f(x)的 定义域为D，f（x）集合D上有定义。 如果存在数K1，使得 f(x)≤K1对任意x∈D都成立，则称函数f(x)在D上有上界。 反之，如果存在数字K2，使得 f(x)≥K2对任意x∈D都成立，则称函数f(x)在D上有下界，而K2称为函数f(x)在D上的一个下界。 如果存在正数M，使得 |f(x)|≤M对任意x∈D都成立，则称函数在D上有界。如果这样的M不存在，就称函数f(x)在D上无界。

定义三：保号性

设数列 $ { \left\{ {x\mathop{{}}\nolimits_{{n}}} \right\} }$ 存在极限a，且a>0(或a<0)，则存在正整数N,当n>N时，有 $ { { {x\mathop{{}}\nolimits_{{n}}}} }$ >0 （或 $ { { {x\mathop{{}}\nolimits_{{n}}} } }$ < 0）

使用： ${\mathop{{{{\mathop{{lim}}\limits_{{n- > \infty }}\mathop{{x}}\nolimits_{{n}}=a\text{ } > 0}\text{ }\text{ }\text{ }= > \text{ }\mathop{{x}}\nolimits_{{n\text{ }}} > 0\text{ }\text{ }\text{脱}\text{帽}\text{法}}}}\nolimits_{{}}}$

推论：如果数列 $ { \left\{ {x\mathop{{}}\nolimits_{{n}}} \right\} }$ 从某项其有${\mathop{{x}}\nolimits_{{n}} \ge 0\text{ }\text{且}\mathop{{lim}}\limits_{{n- > \infty }}\mathop{{x}}\nolimits_{{n}}=a,\text{则}a \ge 0 }$ 戴帽法

极限运算规则

注： 前提是两个函数的的极限存在，复合的极限等于极限的复合

夹逼准则

如果数列 $ { \left\{ {x\mathop{{}}\nolimits_{{n}}} \right\} }$ ， $ { \left\{ {y\mathop{{}}\nolimits_{{n}}} \right\} }$ ， $ { \left\{ {z\mathop{{}}\nolimits_{{n}}} \right\} }$ 满足下列条件

1. ${\mathop{{y}}\nolimits_{{n}} \le \mathop{{x}}\nolimits_{{n}} \le \mathop{{z}}\nolimits_{{n}} \left( n=1,2,3,... \right) }$ 2. ${\mathop{{lim}}\limits_{{n- > \infty }}\mathop{{y}}\nolimits_{{n\text{ }}}=a,\mathop{{lim}}\limits_{{n- > \infty }}\mathop{{z}}\nolimits_{{n}}=a}$

则数列 $ { \left\{ {x\mathop{{}}\nolimits_{{n}}} \right\} }$ 的极限存在，且$\mathop{{lim}}\limits_{{n- > \infty }}\mathop{{x}}\nolimits_{{n}}=a$

单调有界准则

单调有界数列必有极限，即若数列 $ { \left\{ {x\mathop{{}}\nolimits_{{n}}} \right\} }$ 单调增加（减小）且有上界（下界），则 $\mathop{{lim}}\limits_{{n- > \infty }}\mathop{{x}}$ 存在
