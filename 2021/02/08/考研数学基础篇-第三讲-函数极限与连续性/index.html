以点x为中心的任何开区间称为点x的邻域，记作U(x)

$ \delta $ 邻域：设$ \delta $ 是一个整数，则称开区间 ${ \left( {x- \delta ,\text{ }x+ \delta } \right) }$ 为点x的$ \delta $ 邻域，记作U(x, $ \delta $ ), 其中x称为邻域的中心，$ \delta $ 称为领域的半径

函数极限的定义

对于 ${x \to \infty }$ 其对应的 ${ \varepsilon -X}$ 语言为 $ {\mathop{{lim}}\limits_{{x \to \infty }}f \left( x \left) =A \Leftrightarrow \forall \varepsilon > 0,\text{ } \exists X > 0,\text{当}{ \left| {x} \right| } > X\text{时}\text{，}\text{有}{ \left| {f \left( x \left) -A\right. \right. } \right| } < \varepsilon \right. \right. }$

函数极限存在：说明左右极限相等

函数极限的性质

  • 唯一性： 如果极限$\mathop{{lim}}\limits_{{x \to \infty }}f \left( x \right) $ 存在，那么极限唯一

  • 局部有限性： 如果 $\mathop{{lim}}\limits_{{x \to \infty }}f \left( x \left) =A\right. \right. $ ，则存在正常整数M和$ \delta $ ,使得当$0 < { \left| {x-\mathop{{x}}\nolimits_{{0}}} \right| } < \delta $ 时，有${ \left| {f \left( x \right) } \right| } \le M $

  • 局部保号性：如果f(x)->A(x->x。)，且A>0(或A<0),那么存在常数 $ \delta > 0$ ，使得当 $0 < { \left| {x-\mathop{{x}}\nolimits_{{0}}} \right| } < \delta $ 时，有 $f \left( x \left) > 0 \left( \text{或}f \left( x \left) < 0 \right) \right. \right. \right. \right. $

注： 如果$ f \left( x \left) \ge 0 \left( \text{或}f \left( x \left) \le 0 \left) \left( x \to \mathop{{x}}\nolimits_{{0}} \left) \text{且}\mathop{{lim}}\limits_{{x \to \mathop{{x}}\nolimits_{{0}}}}f \left( x \left) =A,\text{则}A \ge 0 \left( \text{或}A \le 0 \right) \right. \right. \right. \right. \right. \right. \right. \right. \right. \right. $
